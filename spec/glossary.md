# Glossary / Słownik roboczy

- **Chronometria Ortogonalna** — formalny system opisu czasu oparty na relacji geometrycznej między wskazaniami kątowymi.
- **Orthogonal Time / Czas Ortogonalny** — stan czasu wyznaczony przez reguły systemu.
- **Base Time / Chwila bazowa** — chwila wejściowa pochodząca z wybranej skali źródłowej.
- **Orth / Orthos** — wartość odchylenia od stanu ortogonalnego w przestrzeni zsymetryzowanej.
- **Centi** — część ułamkowa reprezentacji Orth.
- **Cycle / Cykl** — identyfikator fazy w strukturze 12-godzinnej.
- **Sphere / Sfera** — binarna klasyfikacja topologiczna lub strukturalna; status w v0.1: przejściowy.
- **Trend** — kierunek zmiany wartości Orth; status w v0.1: częściowo eksperymentalny.
