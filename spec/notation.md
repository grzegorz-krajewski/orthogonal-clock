# Notation Guide / Przewodnik notacji

## Domyślna notacja użytkowa
`[<sfera>] C<cykl> O<Oi>.<Ce> <trend>`

Przykład:
`[+] C3 O0.00 ?`

## Notacja skrócona
`<sfera><cykl>:<Oi>.<Ce><trend>`

Przykład:
`+3:0.00?`

## Notacja pełna
`OT[S=<sfera>; C=<cykl>; Orth=<wartość>; Oi=<część_całkowita>; Ce=<centi>; Tr=<trend>]`

## Trend
- `↑` / `up`
- `↓` / `down`
- `=` / `steady`
- `?` / `unknown`
